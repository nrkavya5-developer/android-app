# Shaale-Vikas | Rural School Bridge

A transparent digital bridge connecting rural government schools in India with their global alumni network. Built as a single-page HTML prototype using Alpine.js and Tailwind CSS.

## Features

- **Landing Screen** — Role-based login for Headmaster (Admin) and Alumni
- **Registration Flow** — Context-aware registration form for each role
- **Admin Dashboard** — Key metrics: funds received, problems solved, recent activity
- **Needs Feed** — Browse school needs with search, progress bars, and urgent flags
- **Need Detail** — Full view with description, funding progress, and donate action
- **Donations** — Bottom-sheet modal for alumni to contribute directly
- **Impact Reports** — Before/after photo comparisons of completed projects
- **Hall of Fame** — Leaderboard honoring top donors and supporters
- **Settings** — Profile view, dark mode toggle, notifications, secure logout
- **Dark Mode** — Full theme toggle persisted across screens

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure and semantics |
| Alpine.js (v3.x) | Reactive UI state management |
| Tailwind CSS (CDN) | Utility-first styling |
| Font Awesome 6 | Iconography |
| Google Fonts (Inter) | Typography |

## Quick Start

No build tools or server required. Open the file directly in any modern browser:

```bash
open index.html
```

Or double-click `index.html` in your file explorer.
